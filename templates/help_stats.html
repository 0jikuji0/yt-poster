Comprendre les statistiques

L'outil expose deux niveaux de stats : le suivi local (ce que clipstudio a posté, toujours disponible) et les stats live YouTube (vues, abonnés, likes… récupérées directement via l'API officielle). Cette page explique comment les activer et lire la page Heures de poste.

📊Activer les stats live YouTube

Sur la page Statistiques, sélectionne une chaîne : tu vois ses abonnés, ses vues totales et le détail vue par vue des vidéos postées par l'outil. Ces chiffres viennent en direct de l'API YouTube — deux conditions doivent être réunies.

1

La chaîne doit être connectée en OAuth

Va sur la page de la chaîne et clique « Connecter » pour autoriser l'accès à son compte YouTube. Une chaîne connectée affiche une pastille verte dans la barre latérale. Sans connexion, aucune stat live ne peut être lue.

⚠️ Si l'écran de consentement Google est resté en mode « Test », l'autorisation expire au bout de 7 jours : la chaîne se déconnecte et les stats disparaissent. Passe l'écran en « Production » — voir la doc client_secret.json.
2

L'API YouTube Analytics doit être activée

Dans le projet Google Cloud, l'API YouTube Data API v3 suffit pour poster, mais les stats par jour et l'analyse des heures réclament aussi YouTube Analytics API. Active-la dans API et services → Bibliothèque (étape détaillée dans la doc client_secret.json).

3

Rafraîchir les chiffres

Les stats live sont mises en cache 10 minutes pour économiser le quota API. Le bouton « ↻ Actualiser les stats » en haut du détail d'une chaîne force une nouvelle récupération immédiate.

Astuce : le bloc « Suivi manuel des vues » reste utile même sans stats live — tu peux saisir des relevés de vues à la main sur la page de la chaîne pour tracer une courbe d'évolution.

🕒La page « Heures de poste »

Cette page croise l'heure à laquelle chaque vidéo a été postée avec les vues qu'elle a obtenues, pour repérer les créneaux qui marchent et ceux qui ne rapportent rien. Objectif : régler le planificateur pour ne poster qu'aux bonnes heures.

1

Lire le tableau

Pour chaque heure de la journée, tu vois combien de vidéos y ont été postées et la moyenne de vues récoltées. Une heure « nulle » = des vidéos postées mais sans audience : c'est un créneau à éviter.

Une vidéo n'est comptée qu'à partir de {{ EARLY_MIN_AGE_DAYS }} jours d'ancienneté (réglable via le sélecteur « jours »), le temps qu'elle accumule assez de vues pour que la mesure soit fiable.
2

Agir sur le planificateur

Une fois les bons créneaux identifiés, ajuste la fenêtre horaire de la chaîne (window_start–window_end) sur sa page de réglages, ou réduis le nombre de posts/jour. Le planificateur tirera ses heures aléatoires uniquement dans cette fenêtre.

À retenir : l'analyse des heures s'appuie sur les vues, elle exige donc les mêmes prérequis que les stats live (chaîne connectée + YouTube Analytics API activée).
